
POST /user/authenticate
    = login
    - JSON {email, password}
    -> JSON {valid, token, first_name, last_name}
GET /user/authenticate
    = auth via token
    - Bearer token in Authorisation header
    -> JSON {valid, token, first_name, last_name}
POST /user/create_account
    = register + account setup
    - JSON {email, password, first_name, last_name}
    -> JSON {valid, token, first_name, last_name}
